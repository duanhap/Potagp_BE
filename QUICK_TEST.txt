========================================
QUICK TEST - Copy & Paste Commands
========================================

⚠️ TRƯỚC KHI CHẠY:
1. Thay YOUR_FIREBASE_TOKEN bằng token thật
2. Thay pattern_id=1 bằng ID có trong database của bạn
3. Đảm bảo backend đang chạy: python run.py

========================================
TEST 1: Lấy câu CHƯA THUỘC
========================================

curl -X GET "http://localhost:8002/api/sentences?pattern_id=1&status=unknown" -H "Authorization: Bearer YOUR_FIREBASE_TOKEN" -H "Content-Type: application/json"


========================================
TEST 2: Lấy câu ĐÃ THUỘC
========================================

curl -X GET "http://localhost:8002/api/sentences?pattern_id=1&status=known" -H "Authorization: Bearer YOUR_FIREBASE_TOKEN" -H "Content-Type: application/json"


========================================
TEST 3: Cập nhật câu thành ĐÃ THUỘC (trả lời đúng)
========================================

TEST 4: Tăng số lần SAI (trả lời sai)

========================================
LÀM SAO LẤY FIREBASE TOKEN?
========================================

Cách 1: Từ Android App
- Đăng nhập vào app
- Check Logcat để tìm token
- Token thường có format: eyJhbGciOiJSUzI1NiIsImtpZCI6...

Cách 2: Từ Firebase Console
- Vào Firebase Console > Authentication
- Chọn user > Copy UID
- Dùng Firebase Admin SDK để generate custom token

Cách 3: Test với Postman
- Import collection
- Dùng Firebase Auth để lấy token

========================================
COMMON ERRORS
========================================

401 Unauthorized → Token sai hoặc hết hạn
404 Not Found → pattern_id không tồn tại
403 Forbidden → Không có quyền truy cập pattern này
400 Bad Request → Dữ liệu gửi lên không hợp lệ
